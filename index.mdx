## What is A24z Memory?

A24z Memory is a **Model Context Protocol (MCP) server** that serves as an expert development guide for AI agents. It captures and shares tribal knowledge, best practices, and project-specific insights across your development team.

<Info>
  **Perfect for teams that want their AI agents to:** - Remember complex implementation decisions -
  Follow established patterns and conventions - Access project-specific documentation - Learn from
  past mistakes and solutions
</Info>

## How It Works

The A24z Memory MCP server provides several tools that AI agents can use to interact with your project's knowledge base:

- **`askA24zMemory`** - Query the knowledge base for contextual guidance
- **`create_repository_note`** - Store important insights and decisions
- **`get_repository_tags`** - Discover available categorization tags
- **`get_repository_guidance`** - Access repository-specific documentation guidance

## Quick Installation

### One-Click Setup (Recommended)

**For Cursor Users:**
[![Install MCP Server](https://cursor.com/deeplink/mcp-install-dark.svg)](https://cursor.com/en/install-mcp?name=a24z-memory&config=eyJjb21tYW5kIjoibnB4IC15IGEyNHotbWVtb3J5In0%3D)

This button will automatically configure A24z Memory in your Cursor editor with a single click!

### Manual Installation

For other environments, A24z Memory supports multiple installation methods:

<Tabs>
  <Tab title="Cursor">
    ```bash
    npx a24z-memory install-cursor
    ```
  </Tab>

<Tab title="Claude Code">```bash npx a24z-memory install-claude ```</Tab>

  <Tab title="Manual Installation">
    Add to your MCP configuration file:
    ```json
    {
      "mcpServers": {
        "a24z-memory": {
          "command": "npx",
          "args": ["-y", "a24z-memory"]
        }
      }
    }
    ```
  </Tab>
</Tabs>
