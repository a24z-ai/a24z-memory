# discover_a24z_tools

Discover all available a24z-Memory tools and their capabilities. This tool provides a comprehensive overview of what tools are available and how to use them effectively.

## Overview

The `discover_a24z_tools` tool serves as a self-documenting feature that helps users understand the full range of capabilities available in the a24z-Memory system. It provides detailed information about each tool including descriptions, use cases, parameters, and examples.

<RequestExample>
```json
{
  "category": "repository"
}
```
</RequestExample>

## Parameters

| Parameter  | Type   | Required | Description                                                                             |
| ---------- | ------ | -------- | --------------------------------------------------------------------------------------- |
| `category` | string | No       | Filter tools by category: "all", "knowledge", "repository", "guidance" (default: "all") |

## Tool Categories

### Knowledge Tools

Tools focused on searching, retrieving, and analyzing existing knowledge:

- **askA24zMemory** - Query the knowledge base for contextual guidance
- **find_similar_notes** - Find notes with similar content
- **get_repository_note** - Retrieve specific notes by ID

### Repository Tools

Tools for managing and organizing the knowledge base:

- **create_repository_note** - Create new notes and documentation
- **delete_note** - Remove notes from the knowledge base
- **merge_notes** - Combine related notes
- **review_duplicates** - Analyze for duplicate content
- **check_stale_notes** - Find notes with outdated references

### Guidance Tools

Tools for understanding system capabilities and getting help:

- **discover_a24z_tools** - Discover available tools (this tool)
- **get_repository_tags** - Get available tags and categorization
- **get_repository_guidance** - Get repository-specific guidance
