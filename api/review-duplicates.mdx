# review_duplicates

Analyze the knowledge base to identify duplicate or very similar notes that could be consolidated.

## Overview

The `review_duplicates` tool scans the repository for duplicate or highly similar notes, helping maintain a clean and organized knowledge base.

<RequestExample>
```json
{
  "directoryPath": "/Users/project/my-app",
  "similarityThreshold": 0.9,
  "groupBy": "content"
}
```
</RequestExample>

## Parameters

| Parameter             | Type   | Required | Description                                       |
| --------------------- | ------ | -------- | ------------------------------------------------- |
| `directoryPath`       | string | Yes      | Repository path to analyze                        |
| `similarityThreshold` | number | No       | Similarity threshold 0-1 (default: 0.85)          |
| `groupBy`             | string | No       | Group by 'content' or 'tags' (default: 'content') |

## Response Format

```json
{
  "success": true,
  "duplicateGroups": [
    {
      "groupId": "group-001",
      "similarity": 0.95,
      "notes": [
        {
          "noteId": "note-1734567890123-abc123def",
          "title": "JWT Authentication Setup",
          "createdAt": "2024-01-15T10:30:00Z"
        },
        {
          "noteId": "note-1734567890123-xyz789ghi",
          "title": "Setting up JWT Auth",
          "createdAt": "2024-01-16T14:20:00Z"
        }
      ],
      "recommendation": "Merge into single comprehensive JWT guide"
    }
  ],
  "totalDuplicates": 3,
  "totalGroups": 2
}
```

- Clean up duplicate documentation
- Consolidate similar patterns
- Maintain knowledge base quality
