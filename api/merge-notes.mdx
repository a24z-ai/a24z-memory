# merge_notes

Combine multiple related notes into a single, comprehensive note to consolidate information and eliminate redundancy.

## Overview

The `merge_notes` tool helps consolidate related information by combining multiple notes into one comprehensive note.

<RequestExample>
```json
{
  "noteIds": ["note-1734567890123-abc123def", "note-1734567890123-xyz789ghi"],
  "directoryPath": "/Users/project/my-app",
  "newNote": {
    "note": "Comprehensive guide to authentication patterns combining JWT and OAuth approaches",
    "tags": ["authentication", "jwt", "oauth", "patterns"],
    "type": "pattern"
  }
}
```
</RequestExample>

## Parameters

| Parameter         | Type     | Required | Description                                        |
| ----------------- | -------- | -------- | -------------------------------------------------- |
| `noteIds`         | string[] | Yes      | Array of note IDs to merge                         |
| `directoryPath`   | string   | Yes      | Repository path                                    |
| `newNote`         | object   | Yes      | Content for the merged note                        |
| `deleteOriginals` | boolean  | No       | Delete original notes after merge (default: false) |

## Use Cases

- Consolidate duplicate information
- Combine related patterns or decisions
- Create comprehensive guides from fragmented notes
